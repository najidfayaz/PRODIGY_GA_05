# PRODIGY_GA_05
Neural Style Transfer

Task Overview
This project is part of the Generative AI Internship at Prodigy Infotech.
The objective is to apply the artistic style of one image to the content of another using Neural Style Transfer.

---

Technique Used
- Neural Style Transfer (NST)
- Pre-trained VGG-19 CNN

---

Implementation
- Content and style images are processed
- Feature representations extracted using VGG-19
- Content and style losses are optimized
- A stylized image is generated

---

Output
The final stylized image is saved as `stylized_output.png`.
